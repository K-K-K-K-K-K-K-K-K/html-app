# Mockup Website Project

This project contains simple HTML and CSS mockup pages for a basic website. It demonstrates how to create different pages (Home, Product, Map, Contact) with navigation between them. Bootstrap 5 is used in some pages for styling and layout.

## 📂 Project Structure

- `index.html` — Home page (entry point of the site)
- `product.html` — Product mockup page with Bootstrap 5 styling
- `map.html` — Map mockup page using an iframe (Google Maps embed)
- `contact.html` — Contact page with a form
- `assets/` — (Optional) folder for images, CSS, or other static files

## 🚀 Features

- **Responsive design** using plain CSS and Bootstrap 5.
- **Navigation buttons** linking back to the home page (`index.html`).
- **Contact form** with fields for name, email, subject, and message.
- **Map mockup** using an iframe from Google Maps.
- **Product page** with placeholder image, description, and styled buttons.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (via CDN)

## ▶️ How to Run

1. Clone or download the project files.
2. Open `index.html` in your browser.
3. Use the navigation buttons to explore:
   - Product page (`product.html`)
   - Map page (`map.html`)
   - Contact page (`contact.html`)

## 📌 Notes

- The map iframe currently uses a Google Maps embed link. You can replace the `src` with your own location.
- The contact form is static — to make it functional, connect it to a backend service or form handler.
- Product images are placeholders (`via.placeholder.com`). Replace them with real product images as needed.
